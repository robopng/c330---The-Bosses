## Adaptive AI Arena Fighter (Cycle 1 Prototype)

**Project Adaptive Shadow** is a 2D, top-down, single-player arena fighter. The game's core innovation is a central AI boss designed to **observe, learn, and adapt** its combat strategy to counter the player's unique fighting style.

This repository contains the deliverables for **Cycle 1**, which focuses on establishing the core combat mechanics and the essential data pipeline for future AI development.

---

## üéØ Cycle 1 Deliverables (The Prototype)

The current build (Cycle 1) is a **Proof-of-Concept** and includes the following fully completed features:

| Feature | Description | Status |
| :--- | :--- | :--- |
| **Core Combat** | Player movement, Melee Attack, and basic collision detection. | **Complete** |
| **Data Backbone** | **Critical Feature:** A robust logging system that captures and saves all player combat actions (position, timing, attack type) for future AI training. | **Complete** |

## üöÄ Getting Started

### Prerequisites

* [Unity Hub](https://unity.com/download)
* Unity Editor **202X LTS** (Check project files for exact version)
* A standard keyboard and mouse.

### Installation & Run

1.  **Clone the Repository:**
    ```bash
    git clone [Insert Repository URL Here]
    ```
2.  **Open in Unity:**
    * Open **Unity Hub**.
    * Click **"Add"** and select the root directory of the cloned project.
    * Open the project using the required Unity Editor version.
3.  **Run the Demo Scene:**
    * Navigate to the `Assets/Scenes` folder.
    * Open the `Game_Arena_Scene.unity` file.
    * Press the **Play** button in the Unity Editor.

### Controls

| Action | Key |
| :--- | :--- |
| Move | **W, A, S, D** |
| Melee Attack | **Left Mouse Button** |
